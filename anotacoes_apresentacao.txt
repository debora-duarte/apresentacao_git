Comandos a seguir

-- git init
-- git status
- Criar <arquivo> * Qualquer piadinha
-- git status * Falar sobre o status do repositório
-- git add <arquivo>
-- git status * Falar sobre os arquivos a serem comitados
-- git commit -m "Mensagem"
-- git log
-- git remote add origin <url_repositorio> * Comentar sobre Github x Stash
-- git push -u origin master
- Incluir um arquivo no Github pelo site
-- git pull origin master
-- git status
- Alterar o arquivo
-- git add <arquvio>
-- git reset <arquivo>
-- git checkout -- <arquivo>
-- git branch <nova_feature>
-- git branch
-- git checkout <nova_feature>
- Crio um arquivo nova_feature
-- git add <nova_feature>
-- git commit -m "Comitando nova feature"
- Alterar arquivo
-- git add <nova_feature>
-- git commit -m "Mais uma parte da implementação"
-- git checkout master - Falar do pull request?
-- git merge <nova_feature>
-- git branch -d <nova_feature>
-- git push
